# Pizza-Sales-Report

##  Pizza Sale insights Dashboard

Pizza Sales Analysis | Power BI & SQL | End-to-End Data Analytics Project

Executed full-cycle analytics project analyzing pizza sales data. Built SQL-based KPIs (revenue, orders, AOV), sales trends, and product performance metrics. Created interactive Power BI dashboards showing category/size insights and top/bottom pizzas. Improved decision-making by highlighting revenue drivers and low-performing items.
